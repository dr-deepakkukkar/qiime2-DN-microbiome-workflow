# qiime2-DN-microbiome-workflow
QIIME2 scripts and workflow used in the manuscript titled "A 16S rRNA-based meta-analysis of gut microbiota in diabetic nephropathy using QIIME2 and publicly available NGS datasets". This repository specifically contains the generic commands used for downloading sra files using sra-toolkit and QIIME2 scripts used for imporing the fastq files, denoising (DADA2), taxonomy assignment, diversity analysis, and visualization.

- qiime2.2024.2-
